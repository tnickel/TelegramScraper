📖 Anleitung: Wie bekomme ich einen OpenRouter API-Key?
  1. Gehe auf die OpenRouter Website (openrouter.ai) und melde dich an (z. B. mit deinem Google-Konto).
  2. Klicke oben rechts auf dein Profil-Icon und wähle "Keys" (oder gehe direkt zu openrouter.ai/keys).
  3. Klicke auf "Create Key", gib dem Key einen Namen (z. B. "Forex-Dashboard") und klicke auf "Create". Kopiere den Key sofort.
  4. Füge den Key oben ein, wähle ein Modell (z. B. Llama 3 8B) und klicke auf "Speichern". Die kostenlosen Modelle benötigen kein Guthaben!
